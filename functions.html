setTimeout( )

SetTimeout( ) expects you to pass two things into it. First a callback and second a number of milliseconds to delay the function.

console.log('Hello!')
setTimeout(( ) => {
console.log('Are you still there?')
}, 3000)

Before our arrow function, we have a console.log( ) that prints the string 'Hello' to the console. The function then runs, waits 3 seconds, and our second console.log is printed 'Are you still there?'.

setInterval( )

setInterval( ) is similar to setTimeout( ). It will call a callback every X number of milliseconds.

setInterval(() => {
console.log(Math.random( ))
}, 2000)

The above functionwill print a random number to the console every two 2000 milliseconds, because we have set an interval. To stop the program, we can save the function as a variable and run a clearInterval( ) and pass in the name of the variable.

const id = setInterval(() => {
console.log(Math.random( ))
}, 2000)

// output

0.8155799065531959
0.042167788554137386
0.1133141252213421
0.4790311369150022
clearInterval(id)
