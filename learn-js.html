What is javaScript?

JavaScript is well known as a scripting language used for web pages,but this is not all javaScript is. It's also a fully functional compiled programming languages. Many non-browser environments also use javaScript, such as Node.js, Apache CouchDB and Adobe Acrobat.

Primitive Types

We are going to start by looking at some of the basic building blocks of javaScript. primitive data types are just one of several data types in javaScript. primitive types consist of:

  • Number
  • String
  • Boolean
  • Null
  • Undefined
  • Symbol
  • BigInt

All of these are types that we can store in javaScript e.g. a title on a youtube video would be stored in a string, or the number of likes a video has would be stored in javaScript as a number.

Array Methods

Array methods accept a function as their argument.

Functions

A function is a fundemental building block in javaScript. A function is similar to a procedure, something to be carried out. A set of statements that perform a task.

Arrow Functions

An arrow function is a syntactically compact alternative to a regular function.
